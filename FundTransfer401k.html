Funds Transfer - 401(k)

EE and ER 401(k) account are funded weekly, under most circumstances on Wednesday. Sometimes Tom will request that an out of cycle transfer to close the calendar month.

Processing a 401(k) Funds Transfer with Squee

Here are the steps for sending a funding request to Prudential Retirement. You must have authorization to send secure mail to Payroll Express as well as access the Prudential Sponsor Center website.

Instructions are color coded:
Instructions or steps to be done in LUMS look like this.
Instructions or steps to be done in Squee look like this.

  1. Create a new Transfer in Squee
    • Use the form navigator to go to Prudential
    • Click on the Add Date button.
      1. Squee will ask you to confirm a transfer date, you can change it if you need to.

  2. Add ER batches that are ready to transfer.

    • Click on the ER Log tab.
    • Double click on batches in the Pending list box to move them to the Assigned list box, which adds it to the transfer. The transfer total will also update.
      1. Squee will let you know with a pop-up message if all of the reports in the batch are being added to this transfer.

  3. Add EE batches that are ready to transfer

    • Click on the EE tab.
    • Follow the same steps as adding ER batches.

  4. Check that RA Edit workspace is empty

    • In LUMS go to Receipts & Adjustments /
    • Keystrokes: 4 => {Enter}
    • If the workspace is empty, nothing will happen. If you get the ? prompt, there is something already in the workspace and you need to clear it.
       Really, when nothing happnes that's a good thing.

  5. Check that Squee is in balance with LUMS

    • In LUMS run the Fund Detail Listing for the 401(k) fund pending money.
    • Check the LUMS total against the Squee total.
    • If Squee and LUMS are in agreement continue with process. If not follow the Pru Transfer Troubleshooting Guide

  6. Run the Fund Detail Listing

    1. In LUMS go to Fund Detail Listing /
    2. Keystrokes: 4 => {Enter}
    3. A .pdf of the report will open. The report total must match the transfer total.
  7. Run file build
    1. In LUMS go to Fund Detail Listing /
    2. Keystrokes: 4 => {Enter}
    3. A .pdf of the report will open. The report total must match the transfer total.
    4. In Squee input the EE $ total, ER $ total and ER hours total. They all have a place on the Details tab.
  8. Run Receipts and Adjustments Edit
    1. In LUMS go to Fund Detail Listing /
    2. Keystrokes: 4 => {Enter}
    3. A .pdf of the report will open. The report total must match the transfer total.
  9. Run Receipts and Adjustments Update
    1. In LUMS go to Fund Detail Listing /
    2. Keystrokes: 4 => {Enter}
    3. A .pdf of the report will open. The report total must match the transfer total.
  10. Run teh Fund Detail Listing again
    1. In LUMS go to Fund Detail Listing /
    2. Keystrokes: 4 => {Enter}
    3. A .pdf of the report will open. The report total must match the transfer total.
